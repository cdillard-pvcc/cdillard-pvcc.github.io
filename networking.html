TERM DEFINITION
Network Classifications
  • LAN
  • MAN
  • WAN
  
  • collection of computers in a single building or building complex
  • medium size network such a community
  • world network links machines over great distances
Network Models
  • bus network
  • star network
  
  • machines connected to common communication line called the bus
  • one machines serves as central focal point to which all other machines are connected
Protocols
  • CSMA/CD Ethernet
  • CSMA/CA wifi
  
  • Carrier Sense, Multiple Access with Collision Detection - the protocol that gives the right to tranmit a message on a bus network - dictates that each message is broadcast to all machines on the bus
  • Carrier Sense, Multiple Access with Collision Avoidance - the protocol designed to avoid collisions - when collisions occur, the message must be retransmitted
Combining Networks
  • repeater
  • bridge
  • switch
  • router
  • gateway
  
  • device that passes signals between two busses using amplification without considering the meaning of the signals
  • more complex and more efficeint repeater that checks destination address and forwards a message only when destined for the computer on other side
  • bridge with multiple connections
  • special purpose computers used for forwarding messages that provides links between networks - they maintain a forwarding table for direction of messages
  • the point at which one network is linked to an internet
Client/Server Model

  • software system that controls a network infrastructure that consists of software systems that execute processes on different computers

World Wide Web
  • URL
  • client-side activites
  • server-side activities
  
  • Unifrom Resource Locator - unique address given to locate and retrieve documents on the web - has information need by a browser to contact the proper server
  • activity required by a browser or web server for a client to interact with the page or user
  • activity required by a browser or web server for a server to interact with the page or user
Internet four-layer software
  • application layer
  • transport layer
  • network layer
  • link layer
  
  • software units that use clients and servers that use Internet communication to carry out their tasks
  • used to send and recieve messages over the Internet - divides long messages into smaller segments
  • decides in which direction a packet should be sent at each step through the path along the Internet
  • transfers the packet - deals with the communication details of particular to the network
Internet is:
  • packet-switching network
  • fault tolerant
  • redundant
  
  • the internet is a packet-switch system which digital data is sent by breaking data into packets which contain data and control information
  • ability of a system (network) to continue operating without interuption when components fail
  • there is a backup for delivery when congestion or failures occur - increases reliability and helps scale to more people and devices

  • spyware

  • autonomous program that transfers itself througha network, taking up residnece in computers and forwarding copies to other computers
  • enters a computer system disguised as a desirable program - they are often in form of email attachment
